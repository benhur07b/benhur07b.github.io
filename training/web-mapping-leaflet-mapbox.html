Web Mapping & Web GIS (Basic Web Mapping with Leaflet & Mapbox)

Who is it for

People who want to learn how to create simple web maps using Leaflet & Mapbox. Prior programming experience is recommended but not required.

What to expect

Introductory course on how to build simple web maps with free and open source libraries Leaflet and Mapbox. At the end of the training, you should be familiar with the anatomy of a web mapping application and be able to create your own simple web maps using Leaflet and Mapbox.

Duration

Approximately 8 hours (1 day)

Topics covered

Anatomy of a web map & web mapping applications
Creating web maps with Leaflet
Creating web maps with Mapbox

Fees

PhP 3,000.00 / person
