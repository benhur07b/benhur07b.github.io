Spatial Databases (Introduction to Spatial SQL)

Who is it for

People who want to learn how to utilize the power of SQL in working with spatial data. Previous GIS and SQL experience is recommended but not required.

What to expect

Introductory course to using Structured Query Language (SQL) for working with spatial data. By the end of the training, you should be familiar with performing common spatial analysis processes with SQL in SpatiaLite and QGIS. You should be able to explain relational database concepts, execute basic SQL commands, and perform spatial SQL queries like joins, interesects, among others.

Duration

Approximately 8 hours (1 day)

Topics covered

Spatial Databases
SQLite and SpatiaLite
Data Management in SQLite
SQL Queries
Attribute Queries
Geometry Queries
Viewing Spatial Tables & Views in QGIS
QGIS DB Manager
