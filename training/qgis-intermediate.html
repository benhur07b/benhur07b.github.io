QGIS Training (Intermediate)

Who is it for

Users with experience working with QGIS & those who want to transcend to power-user status.

What to expect

By the end of the training, you should be comfortable with using QGIS to query and analyze geospatial data from a database; utilizing processing providers like GRASS and LAStools; leveraging Python and the Graphical Modeler for complex processing algorithms; creating advanced symbologies; using Expressions in QGIS; and automating map and report generation using Atlas and Reports.

Duration

Approximately 24 hours (3 days)

Topics covered

QGIS and Databases
QGIS and other Processing Providers
Advanced Styling Techniques
Geometry Generators
Advanced Expressions
QGIS and Python
QGIS Graphical Modeler
QGIS Atlas & Reports

Certificates

The course is vetted by the QGIS community. If you opt for a certificate, you will receive a digital certificate with a unique certificate ID. The certificate fee is donated back to the QGIS project.

Fees

PhP 8,500.00 / person (without Certificate)
PhP 10,000.00 / person (with Certificate)
