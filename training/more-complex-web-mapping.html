Web Mapping & Web GIS (Building Web Mapping Applications)

Who is it for

People who want to learn how to create more advanced and complex web maps using Leaflet & Mapbox. Prior programming experience (esp. JavaScript) is required.

What to expect

Builds on the Basic Web Mapping course and provides examples, use cases, and best practices on how to create advanced and complex web maps using Leaflet & Mapbox.

Duration

Approximately 16 hours (2 days)

Topics covered

Geolocation
Geocoding
Routing
Complex map interactions
Geoprocessing with Turf.js
Datasets and Tilesets using Mapbox Studio
Data-driven interactions with Mapbox Expressions
