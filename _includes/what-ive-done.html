What I've done

STOMP COVID-19
The COVID-19 Status Tracking, Overview, and Mapping Project (STOMP COVID-19) is a COVID-19 spatial and data platform developed to provide open, easy-to-use, and granular COVID-19 maps and data in the Philippines.
It uses free and open technologies like Mapbox, CARTO, Jekyll, and Airtable.

Open Hazards PH
A QGIS plugin that lets you load geospatial hazard data created by various agencies in the Philippines.
